How to Export ChatGPT to Markdown (Free, Open-source)

Follow this guide to install the ChatGPT Exporter userscript and save any conversation as clean Markdown. No extensions, no external services — it all runs in your browser.

Prefer PDF? Jump to the PDF tutorial. Need a high-level overview? Visit the landing page.

What you need

  • A modern desktop browser (Chrome, Edge, Firefox, Brave, or Arc).
  • A userscript manager such as Tampermonkey or Violentmonkey.
  • Access to the ChatGPT conversation you want to export.

Step-by-step

1
Install a userscript manager

Visit the Tampermonkey or Violentmonkey website and install the extension for your browser. Open their dashboard to confirm the install.

2
Add the Markdown userscript

Open the ChatGPT Chat Exporter — Markdown listing on GreasyFork and click “Install”. Your manager shows a confirmation prompt.

3
Open ChatGPT and refresh

Head to chat.openai.com, open any conversation, and refresh once. You’ll see a new “Export Markdown” button in the header.

4
Export your conversation

Click the button, pick a file name, and the exporter will download a .md file to your machine. Code blocks, tables, citations, and images (as placeholders) stay intact.

Verify your Markdown

Open the file in your editor or note-taking app. Headings, ordered lists, code fences, and answer citations are formatted for Git, Obsidian, or Notion. Long conversations chunk cleanly with Markdown heading anchors.

Troubleshooting tips

  • No button? Confirm the userscript is enabled and refresh the page. Some browsers require you to allow the domain.
  • Long chats truncate? Expand all messages before exporting. The exporter walks the DOM and needs each message visible.
  • Images? They are referenced as placeholders. Download originals from the inline viewer if needed.
  • Console method? Use the Markdown snippet in the docs for environments where extensions are blocked.
