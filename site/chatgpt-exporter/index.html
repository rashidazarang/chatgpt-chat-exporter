Why choose this over extensions?

Extensions demand broad permissions and run background scripts. ChatGPT Exporter executes client-side, keeps your conversations local, and focuses on output quality.

Runs in your browser

No installs, no extra processes. Userscripts and console snippets execute in the page you already have open.

Minimal footprint

No background listeners, service workers, or network calls. Review the exact code before you run it.

Cleaner outputs

Markdown keeps headings, code blocks, tables, and citations intact. PDFs use a tuned layout with dark-mode aware typography.

Multiformat ready

Markdown, HTML, PDF, and Gemini Markdown — out of the box. Additions land via GitHub releases.

Userscript method (recommended)

  1. Install a userscript manager such as Tampermonkey or Violentmonkey.
  2. Add the Markdown or PDF script from GreasyFork.
  3. Open any ChatGPT conversation and click Export.

Need a walkthrough? Follow the Markdown tutorial.

Console method (zero install)

  1. Open your browser developer tools and switch to the Console tab.
  2. Paste the single-file exporter script from GitHub.
  3. Download Markdown or PDF instantly.

Great for quick exports or restricted environments. See the docs for the latest snippet.

Frequently asked questions

Security, support, and day-to-day usage.

Does the exporter send my chats anywhere?

No. Everything runs in your browser tab. The userscripts and console snippets never transmit data — verify it directly on GitHub.

Do I need a paid ChatGPT plan?

No. The exporter works on free and paid tiers. You only need access to the conversation in your browser.

Will this work with Claude or other models?

Today the focus is ChatGPT and Gemini. Contributions for other providers are welcome — open an issue on GitHub.

How do updates ship?

Releases publish to GitHub and GreasyFork. Install once, and your userscript manager prompts you when new versions are available.

Why two userscripts instead of one?

Markdown and PDF pipelines are tuned separately. Pick the format you need — or install both to toggle from the toolbar.
